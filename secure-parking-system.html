🛡️ ACCESS CONTROL MATRIX & ADMINISTRATION

Access Control List (ACL)

This matrix defines permissions for different user roles on system objects.

Subject (Role) Book Slot Cancel Booking View All Bookings Manage Users Access Logs Override Bookings
Regular User ✓ Allow ✓ Own Only ✗ Deny ✗ Deny ✗ Deny ✗ Deny
VIP User ✓ Allow ✓ Own Only ✓ Read Only ✗ Deny ✗ Deny ✗ Deny
Administrator ✓ Allow ✓ All ✓ Full Access ✓ Allow ✓ Allow ✓ Allow

Policy Justification:

Regular Users: Can book and manage their own slots to ensure fair access. Cannot view others' bookings to protect privacy.

VIP Users: Have additional read access to all bookings for planning purposes but cannot modify others' reservations.

Administrators: Full access to manage system, override bookings in emergencies, and monitor security logs for system integrity.
